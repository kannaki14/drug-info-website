About MediGuide

MediGuide is a beginner-friendly drug information website that demonstrates a full-stack setup with HTML, CSS, JavaScript, Node.js, Express, and MongoDB.

It provides quick access to common drug usage details, dosage information, side effects, warnings, and a simple interaction checker using a demo dataset.

Note: This website is for educational purposes only and is not a substitute for professional medical advice.
